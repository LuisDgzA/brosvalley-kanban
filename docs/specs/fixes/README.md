# Fix Specs — BrosValley CRM

Fecha: 2026-05-13

## Proposito

Esta carpeta contiene specs cortos para:

- bugs;
- detalles visuales;
- ajustes de UX;
- correcciones tecnicas acotadas.

## Regla de uso

Usar un fix spec cuando el cambio:

- no introduce una capacidad nueva grande;
- afecta una parte concreta del producto;
- puede describirse con problema, resultado esperado y criterios de aceptacion.

## Convencion sugerida
